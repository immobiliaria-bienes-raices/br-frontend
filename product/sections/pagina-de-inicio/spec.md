# Página de Inicio Specification

## Overview
Landing page styled after Engel & Völkers with an elegant, minimalist design. Presents the brand identity, featured properties in a carousel, value proposition, and clear calls-to-action to drive users to explore properties or contact the team.

## User Flows
- User lands on homepage and sees hero section with brand message and CTA
- User browses featured properties carousel (swipeable on mobile)
- User clicks property card to navigate to property details
- User clicks "Explorar Propiedades" CTA to view full catalog
- User scrolls to learn about services offered
- User clicks contact CTA to reach out

## UI Requirements
- Hero section with full-width background image, elegant headline, and CTA button
- Featured properties carousel with property cards showing image, location, price, and area
- Value proposition section highlighting key benefits (professional service, insurance partnerships, full delegation)
- Services overview section with icons/cards for each service
- Contact CTA section with strong call-to-action
- Logo displayed in header from `/assets/logo.png`
- Sample property images from `/mocks/` directory
- Responsive design with mobile-first approach
- Smooth transitions and hover states
