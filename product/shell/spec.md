# Application Shell Specification

## Overview
Engel & Völkers-style navigation shell for the Bienes Raíces: Inmobiliaria real estate platform. Clean and professional design with horizontal top navigation.

## Navigation Structure
- **Logo** → Página de Inicio (home)
- **Inicio** → Página de Inicio
- **Propiedades** → Catálogo de Propiedades
- **Servicios** → Servicios
- **Contacto** → Contacto

## Layout Pattern
Top Navigation — Fixed horizontal navigation bar at the top with:
- Logo and brand name on the left
- Navigation links center/right aligned
- Clean design with ample white space

## Responsive Behavior
- **Desktop:** Full horizontal navigation visible, logo on left, links on right
- **Tablet:** Condensed navigation, possible hamburger menu
- **Mobile:** Hamburger menu with slide-out drawer

## Design Tokens Applied
- **Background:** #F5F2F2 (warm white)
- **Accent/Active:** #77333B (Pantone 19-1629 TCX Ruby Wine)
- **Text/Borders:** slate
- **Typography:** Manrope

## Design Notes
- Style inspired by Engel & Völkers
- Fixed header on scroll
- Smooth hover state transitions
- Prominent logo to reinforce brand
- Minimalist footer with contact information
